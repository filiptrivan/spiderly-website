## Install Prerequisites

Before getting started with Spiderly, make sure you have the following prerequisites installed:

- [Visual Studio Code](https://code.visualstudio.com/download)
- [.NET 9.0](https://dotnet.microsoft.com/en-us/download)
- [PostgreSQL](https://www.postgresql.org/download/) or [SQL Server](https://www.microsoft.com/en-us/sql-server/sql-server-downloads)
- [Node.js](https://nodejs.org/en/download/)
- [C# Dev Kit](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csdevkit)
- Run the Spiderly CLI and EF Core CLI installation commands from any terminal location:
  ```bash
  dotnet tool install -g Spiderly.CLI
  dotnet tool install -g dotnet-ef --version 9.0.1
  ```

## Initialize the App

Open the new VS Code window in the folder **where you want your app to be located** and run this command in the VS Code terminal (*Terminal -> New Terminal*, or press `` Ctrl + Shift + ` ``):

```bash
spiderly init
```

## Start the App

Press `F5` to start the app.

## Register the User

Use the UI of your generated app to register a new user via email.

<Callout>
In development mode, if you haven't configured email services yet, the verification code will appear in a notification popup in the upper-right corner of the screen. Simply copy and paste this code to complete the registration.

For production, see [Set Up Emailing](/docs/set-up-emailing) to configure a real email provider.
</Callout>
